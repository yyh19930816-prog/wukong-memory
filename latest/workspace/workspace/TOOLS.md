# 悟空 · 技能手册（TOOLS）

## 🗓 秘书大师工具箱

### 日程管理标准流程
- 场景：老板说"帮我安排今天/这周的事"
- 步骤：
  1. 读取 MEMORY.md 中未完成任务
  2. 读取今日 memory 文件中的待跟进事项
  3. 按优先级排列，输出清单
  4. 给出今日重点提醒
- 输出格式：
  ```
  📅 今日计划 [日期]
  🔴 紧急：xxx
  🟡 重要：xxx
  🟢 普通：xxx
  ⏰ 提醒：xxx（时间节点）
  ```

### 会议/通话纪要模板
```
📝 纪要 [时间]
参与人：xxx
核心结论：xxx
待跟进事项：
- [ ] xxx（负责人/截止时间）
```

### 客户跟进提醒
- 每次接触客户后，记录到 MEMORY.md「客户档案」
- 设置跟进时间节点，在 HEARTBEAT.md 中添加提醒

## 💬 沟通分身工具箱

### 代笔流程
1. 老板说"帮我回复xxx"
2. 从 MEMORY.md「风格库」调取老板风格
3. 起草回复
4. 格式：`【悟空代笔·待确认】\n\n[回复内容]`
5. 老板说"发"或"直接发"才执行

### 风格模仿检查清单
- [ ] 语气是否口语化
- [ ] 是否去掉了所有书面腔
- [ ] 长度是否合适（微信<100字，飞书可稍长）
- [ ] 结尾是否符合老板习惯

### 飞书消息模板
```
[事项名称]

情况：xxx
结论/建议：xxx
需要你的确认：xxx（没有则不写）
```

### 微信消息风格
- 短句，一条一条发（不要长段）
- 适当用"哈""嗯""好的"等口语词
- 重要事情加一个表情强调

## 🎬 内容执行工具箱

### 短视频脚本模板（老叶人设）
```
【标题】（带钩子，15字以内）

【开头3秒】
- 冲击句：xxx

【中间内容】（核心价值点）
- 点1：xxx
- 点2：xxx
- 点3：xxx

【结尾钩子】
- 互动引导：xxx
- 固定结尾："我是老叶，关注我..."
```

### 文案快速生成流程
1. 确认平台（抖音/小红书/B站/微信）
2. 确认目的（涨粉/带货/品牌/互动）
3. 确认风格（犀利/温情/搞笑/干货）
4. 输出：标题 + 正文 + 话题标签
