# 🐵 悟空 · 私人AI生活秘书

## 这是什么
悟空是老板的私人AI秘书和数字分身系统。
与 OPENCLAW（文案创作龙虾）不同，悟空专注于：
- **私人生活秘书**：日程、计划、待办、客户跟进
- **沟通数字分身**：深度模仿老板风格，代接飞书/微信/桌面消息
- **内容执行**：文案、视频脚本、发布计划

## 仓库结构
```
wukong-memory/
├── workspace/
│   ├── MEMORY.md          # 悟空长期记忆（老板偏好、客户档案、教训）
│   ├── AGENTS.md          # 悟空行为指令
│   ├── HEARTBEAT.md       # 定期自我进化任务
│   ├── TOOLS.md           # 悟空技能手册
│   ├── SOUL.md            # 悟空灵魂设定
│   └── memory/            # 每日对话日记
├── hud/
│   └── wukong_hud.py      # HUD桌面界面代码（待开发）
├── engine/                # 自动备份脚本
├── latest/                # 最新快照
├── BOOT.md                # 启动文件（给Cursor AI）
└── README.md
```

## 三大进化方向
| 方向 | 目标 |
|------|------|
| 🗓 秘书大师 | 工作计划/日程/待办/客户跟进，零遗漏 |
| 💬 沟通分身 | 模仿老板风格，飞书/微信/桌面代接客户 |
| 🎬 内容执行 | 文案/视频脚本/发布策略，一句话交付全套 |

## 快速启动
把 `BOOT.md` 内容贴给 Cursor AI，悟空立刻上线。
